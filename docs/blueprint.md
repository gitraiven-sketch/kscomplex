# **App Name**: Kabwata Shopping Complex

## Core Features:

- Property Management: Admins can add, update, and delete property listings, including details like address, rent, and lease terms.
- Tenant Management: Admins can add, update, and remove tenant profiles, associating them with specific properties and lease agreements.
- Payment Tracking: Record and track tenant payments, including the date paid, amount, and payment method. Able to determine which tenant is overdue and who is about to pay their dues.
- Automated Rent Reminders: Configure and schedule automated WhatsApp reminders to tenants about upcoming rent payments, utilizing a tool that reasons about timing.
- Login and Logout: Secure authentication system for admins and tenants with password protection.
- Payment History: Tenants can view their complete payment history, including dates, amounts paid, and any outstanding balances.
- Reporting and Analytics: Generate reports on overdue payments, upcoming payments, and overall financial performance.
- Search Bar: Enable searching of tenants and properties.

## Style Guidelines:

- Primary color: Deep sky blue (#32ADE6) to convey trust and reliability.
- Background color: Very light cyan (#E5F6FD), close in hue to the primary color but highly desaturated, to keep the layout airy and clean.
- Accent color: Purple (#A06CD5) for key actions and visual interest.
- Body and headline font: 'Inter', a sans-serif font providing a modern and clean feel, suitable for both headlines and body text.
- Use a consistent set of icons for navigation and key actions, prioritizing clarity and ease of recognition. Icons should be professional and modern, sourced from a reputable library like Font Awesome or Material Design Icons.
- Implement a clean, card-based layout for properties and tenant information, making it easy to scan and digest data. Ensure consistent spacing and alignment throughout the interface. Use a responsive design framework (e.g., Bootstrap, Tailwind CSS) to ensure optimal viewing across different devices.
- Subtle transitions and animations to enhance user interaction, such as sliding panels or feedback on button presses. Animations should be used sparingly and purposefully to avoid distracting users.
